# GMSHOLDING.GE Landing Page

Ultra-premium futuristic construction and development homepage for **GMSHOLDING.GE**.

## Deployment

This is a static website optimized for standard Hostinger Business hosting.

1. Upload `index.html` and the `assets` folder to `public_html`.
2. Keep the same folder structure so `assets/styles.css` and `assets/app.js` load correctly.
3. Replace the demo form action and Google Maps placeholder when production contact details are ready.
